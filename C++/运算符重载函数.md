运算符重载函数（必须固定命名）：
运算符	对应的函数名
+	    operator+
-	    operator-
*	    operator*
/	    operator/
==	    operator==
<<	    operator<<

例子：
class Complex {
    double real, imag;
public:
    Complex(double r, double i) : real(r), imag(i) {}
    
    Complex operator+(const Complex& other) {
        return Complex(real + other.real, imag + other.imag);
    }
};

// 使用：
Complex c1(1, 2), c2(3, 4);
Complex c3 = c1 + c2;  // 语法直观自然！
