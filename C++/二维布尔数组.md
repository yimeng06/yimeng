vector<vector<bool>> dp(m, vector<bool>(n, false));//创建一个m行n列的二维布尔数组，初始值均为false
最内层：vector<bool>(n,false)
创建一个一维vector：
1、包含 n 个元素
2、每个元素都是 bool 类型
3、全部初始化为 false

外层：vector<vector<bool>> dp(m, ...)
创建一个二维vector（vector的vector）：
1、包含 m 个元素
2、每个元素都是我们第一步创建的 vector<bool>(n, false)
3、所以得到 m × n 的二维数组

大概就是这个意思
如果 m = 3, n = 4：
dp = {
    {false, false, false, false},  // 第0行
    {false, false, false, false},  // 第1行  
    {false, false, false, false}   // 第2行
}

访问方式
dp[i][j] = true;     // 访问第i行第j列
bool value = dp[1][2]; // 获取值
