C++中的new运算符
new运算符
在C++中，new用于在堆（heap）上动态分配内存。它的一般形式是：
new 类型[大小] // 用于数组
new 类型(初始值) // 用于单个对象并初始化
例：
char* name = new char[100];  // 在堆上分配，需要手动释放
// 使用后必须：
delete[] name;  // 手动释放内存

还可以用于创造二维数组：
int n = 3, m = 4;  // 3行4列

// 创建行指针数组
int** ptr = new int*[n];

// 为每一行创建列数组
for (int i = 0; i < n; i++) {
    ptr[i] = new int[m];
}

// 现在可以像普通二维数组一样使用
ptr[0][0] = 1;
ptr[1][2] = 5;

还可以用连续的方法：
int n = 3, m = 4;

// 一次性分配所有内存
int* data = new int[n * m];
int** ptr = new int*[n];

// 设置行指针
for (int i = 0; i < n; i++) {
    ptr[i] = &data[i * m];
}

// 使用方式相同
ptr[1][2] = 5;

这是非连续的也就是第一种
ptr → [int*] → [0,0][0,1][0,2][0,3]
      [int*] → [1,0][1,1][1,2][1,3]  
      [int*] → [2,0][2,1][2,2][2,3]

这是连续的
ptr → [int*] → [0,0][0,1][0,2][0,3][1,0][1,1]...
      [int*] ↗
      [int*] ↗
