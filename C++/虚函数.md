虚函数是在基类中使用关键字virtual声明的成员函数。派生类可以对基类中的虚函数进行重写（override），以实现不同的功能。
没有虚函数的情况
class Base {
public:
    void show() { cout << "Base" << endl; }
};

class Derived : public Base {
public:
    void show() { cout << "Derived" << endl; }
};

int main() {
    Base* ptr = new Derived();  // 基类指针指向派生类对象
    ptr->show();                // 输出 "Base"，不是 "Derived"
    delete ptr;
    return 0;
}
有虚函数的情况
class Base {
public:
    virtual void show() { cout << "Base" << endl; }  // 虚函数
};

class Derived : public Base {
public:
    void show() override { cout << "Derived" << endl; }
};

int main() {
    Base* ptr = new Derived();  // 基类指针指向派生类对象
    ptr->show();                // 输出 "Derived" ✓
    delete ptr;
    return 0;
}
虚函数还可以构成析构函数
class Base {
public:
    virtual void show() {   // 虚函数
        cout << "Base show" << endl;
    }
    virtual ~Base() {       // 虚析构函数
        cout << "Base destructor" << endl;
    }
};

class Derived : public Base {
public:
    void show() override {  // 重写基类的虚函数
        cout << "Derived show" << endl;
    }
    ~Derived() {
        cout << "Derived destructor" << endl;
    }
};

int main() {
    Base* basePtr = new Derived();
    basePtr->show();   // 输出 "Derived show"，因为show是虚函数
    delete basePtr;    // 由于析构函数是虚函数，会先调用Derived的析构函数，然后调用Base的析构函数

}

输出：
Derived show
Derived destructor
Base destructor

可以看到，先调用派生类的析构函数，再调用基类的析构函数。
