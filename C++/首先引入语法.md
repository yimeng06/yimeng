首先引入语法int& refVar;  // 在类中声明一个引用成员变量
必须在构造函数初始化列表中初始化
cpp
class Example {
private:
    int& ref;  // 引用成员变量的声明
    
public:
    // 必须在初始化列表中初始化引用
    Example(int& ex) : ref (ex) { }
    //                         ↑ 这里进行初始化
};//共享数据：多个对象共享同一数据，意思就是例如例子里的ref假设ex在main里面指向了a，那么就相当于ref也指向了a，a改变他也改变他改变a也改变，除非是const
// 实际：ref 绑定到 ex 所绑定的变量
// 相当于：ref 和 ex 都成为同一个变量的别名

int value = 42;

还可以const引用const int& constRef;  // const引用
Const引用1. 引用不可重新绑定
2. 通过引用不能修改值

还有一个双const的
// 返回const引用，并且函数是const的
    const int& getValue() const {
        // 这里不能修改importantValue，因为函数是const的
        return importantValue; // 返回的是const引用，所以外部不能通过这个引用修改importantValue
    }

    // 对比：如果返回非const引用，那么外部就可以修改
    int& getValue() {
        return importantValue; // 返回非const引用，外部可以修改importantValue
    }
// 注意：返回const引用的函数不能修改对象的数据成员，只能读取，但是不能修改对象本身的状态。
