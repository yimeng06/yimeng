在C++中更推荐用const去限定一个符号；其公式为
Const type name = value;
他和#define有大差别例如
#define MAX 10+10

int main() {
    int value = 2 * MAX;  // 期望是2*20=40，但实际是2*10+10=30
    return 0;
}
而使用const则不会：
const int MAX = 10+10;

int main() {
    int value = 2 * MAX;  // 2*(10+10)=40，符合预期
    return 0;
}
•	#define：没有作用域的概念，它从定义点开始到文件末尾都有效（除非被#undef取消）。它不遵循C++的作用域规则，可能会在无意中影响其他部分的代码。
•	const：有作用域，可以定义在全局、命名空间、类内部或函数内部。它遵循相同的作用域规则，不会影响到其他作用域的代码。
#define 是简单的"文字替换"，const 是真正的"类型安全变量"。
