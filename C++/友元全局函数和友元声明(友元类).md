友元全局函数是一个在类外部声明的全局函数，但被类声明为友元（friend）。这意味着该函数可以访问类的私有（private）和保护（protected）成员，尽管它不是类的成员函数。
并且他不受访问控制：可以在private、protected或public区域声明，还可以突破封装：可以访问类的私有和保护成员

例：
class Point {
private:
    double x, y;

public:
    Point(double xVal = 0, double yVal = 0) : x(xVal), y(yVal) {}
    
    // 声明友元全局函数
    friend double distance(const Point& p1, const Point& p2);
    friend void printPoint(const Point& p);
};

// 友元全局函数的定义
double distance(const Point& p1, const Point& p2) {
    // 可以直接访问私有成员 x 和 y
    double dx = p1.x - p2.x;
    double dy = p1.y - p2.y;
    return sqrt(dx * dx + dy * dy);
}

void printPoint(const Point& p) {
    // 可以直接访问私有成员
    cout << "Point(" << p.x << ", " << p.y << ")" << endl;
}
int main() {
    Point p1(1, 1);
    Point p2(4, 5);
    
    printPoint(p1);  // 输出: Point(1, 1)
    printPoint(p2);  // 输出: Point(4, 5)
    
    cout << "Distance: " << distance(p1, p2) << endl; // 输出: Distance: 5
    
    return 0;
}

举一个例子：Complex(double r = 0, double i = 0) : real(r), imag(i) {}；（这是一个构造函数我把他和友元全局函数一起讲了）
Complex
•	这是构造函数的名称，必须与类名相同
•	构造函数在创建对象时自动调用
(double r = 0, double i = 0)
•	参数列表：构造函数接受两个参数
•	double r = 0：第一个参数，类型是double，参数名是r，默认值是0
•	double i = 0：第二个参数，类型是double，参数名是i，默认值是0
•	= 0 表示如果没有提供这个参数，就自动使用0
: real(r), imag(i)
•	这是成员初始化列表
•	real(r)：将成员变量real初始化为参数r的值
•	imag(i)：将成员变量imag初始化为参数i的值
•	逗号分隔多个成员的初始化
•	冒号：用于引入初始化列表
{}
•	构造函数的函数体（这里是空的，因为初始化已经在初始化列表中完成了）
其实就相当于Complex(double r = 0, double i = 0) {
    real = r;
    imag = i;
}

这是友元声明友元
友元类：当一个类将另一个类声明为友元时，友元类可以访问该类的私有和保护成员
class Phone {
private:
    int price;  // 私有成员，外面不能直接访问
    
public:
    Phone(int p) : price(p) {}
    
    // 声明myphone类是友元，给它开后门
    friend class myphone;
};

class myphone {
public:
    void showPrice(Phone &p) {
        cout << p.price;  // ✅ 可以访问Phone的私有成员price！
    }
};

1. 友元类（Friend Class）
cpp
class A {
private:
    int secret;
    
public:
    friend class B;  // B类是我的朋友，可以访问我的私有成员
};
2. 友元全局函数（Friend Function）
cpp
class A {
private:
    int secret;
    
public:
    friend void showSecret(A &a);  // 这个全局函数是我的朋友
};

// 全局函数
void showSecret(A &a) {
    cout << a.secret;  // 可以访问私有成员
}
3. 友元成员函数（Friend Member Function）
cpp
class A {
private:
    int secret;
    
public:
    friend void B::showAsecret(A &a);  // B类的某个成员函数是我的朋友
};
