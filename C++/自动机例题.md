实现一个 myAtoi(string s) 函数，使其能将字符串转换成一个 32 位有符号整数。

函数 myAtoi(string s) 的算法如下：

空格：读入字符串并丢弃无用的前导空格（" "）
符号：检查下一个字符（假设还未到字符末尾）为 '-' 还是 '+'。如果两者都不存在，则假定结果为正。
转换：通过跳过前置零来读取该整数，直到遇到非数字字符或到达字符串的结尾。如果没有读取数字，则结果为0。
舍入：如果整数数超过 32 位有符号整数范围 [−231,  231 − 1] ，需要截断这个整数，使其保持在这个范围内。具体来说，小于 −231 的整数应该被舍入为 −231 ，大于 231 − 1 的整数应该被舍入为 231 − 1 。
class Automaton{
    string state ="start";
        unordered_map<string,vector<string>> table = {
            {"start",{"start","signed","in_number","end"}},
            {"signed",{"end","end","in_number","end"}},  //这个是等待数字状态前面的start已经把符号读完了所以符号位是end
            {"in_number",{"end","end","in_number","end"}},
            {"end",{"end","end","end","end"}}
        };

        int get_col(char c){
            if(isspace(c)) return 0;//检查空白格
            if(c == '+' or c == '-') return 1;
            if(isdigit(c))  return 2;//检查数字
            return 3;
        }
        public:
        int sign = 1; //符号，1表正数，-1表负数
        long long ans = 0;//储存计算结果

        void get(char c){
            //1、更新状态：查表到下一个状态 
            state = table[state][get_col(c)];
            //2、如果是数字状态就处理数字
            if(state == "in_number"){
                ans = ans * 10 + c -'0';//累加数字
                ans = sign == 1? min(ans,(long long)INT_MAX) :min(ans,-(long long)INT_MIN);
                //检查溢出
            }
            //3、如果是符号状态就处理符号
            else if(state == "signed")
            sign = c == '+'? 1:-1;
        }
};

class Solution {
public:
    int myAtoi(string str) {
        Automaton automaton;
        for(char c : str) //"遍历字符串str中的每一个字符，每次循环把当前字符赋值给变量c" 
        {
            automaton.get(c);
        }
        return automaton.sign * automaton.ans;
    }
};

在编程实践中，自动机 ≈ 状态机（指的是同一个东西）
在理论科学中，自动机是一个更大的概念，包含多种类型
大概就是这样一个关系
A[自动机 Automata] --> B[有限状态机 FSM]
    A --> C[下推自动机 PDA]
    A --> D[线性有界自动机 LBA]
    A --> E[图灵机 Turing Machine]
    
    B --> F[确定型有限自动机 DFA]
    B --> G[非确定型有限自动机 NFA]

自动机就是状态机（在编程中）

状态机就是有限状态机（FSM）

这里我的 Automaton 类就是一个状态机
